Assembling a Package Item



Packages, like repositories, are made completely in JSON. For a more complete outline, look at test.package.json provided on GitHub.

"results" is the array that contains the entire package information in objects.

"name" is the name of an item. "file" is what is displayed on the list.
"goto" is the link to download a file. "run" is the JavaScript that is run when something is clicked on. It is advised to leave the other blank when one is in use.
