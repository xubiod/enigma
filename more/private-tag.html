Private Badge Handling

There is a "Private" tag, but nothing is done in the backend. This is a psuedo-guide on how to use it properly.

Handling

The way that private items are handled are dependent on the page loaded into the workspace, whether it's an IP address check or through OAuth methods. Please note that a non-OAuth method for usernames and passwords can be easy to crack.

IP Address Checks

There are a few APIs that can do just that - return the request's public IP address. You want to make it JSON so you can fetch (vanilla JS) or $.getJSON() (jQuery).
This StackOverflow question's first answer has many APIs and jQuery snippets to use.

reCaptchas

Using Google's Captcha is one way to prevent spam and fruadulent requests from bots. You can get more information on the official site.

OAuth

OAuth has become the security standard for APIs, usernames and passwords. Visit the official web site for more information.
